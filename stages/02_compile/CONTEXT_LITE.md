# Stage 02 — Compile (LITE)

*Pre-chewed contract for thrift-ON models (tier B/C). Same job as `CONTEXT.md`,
fewer words, anchors restated. Use this when your card sets context policy ON.*

**Do:** turn the research into a factsheet and an outline. Select facts, fix
their reliability, build the argument, plan one metaphor, mark any visuals.

**READ** (in this order):
1. `../00_brief/output/brief.md` — goal, audience, format, length, D-list,
   `visuals` dial.
2. `../01_research/output/findings.md` + `sources.md` — the F-tagged facts.
3. `_config/style/imagery.md` — for the metaphor.
4. `_config/style/visuals.md` — only if `visuals: auto`/`on`.
5. The audience file named in the brief.

**WRITE `output/factsheet.md`:** keep only facts that serve a D-item. Tag each
`F<n> [S-strong|S-medium|S-weak]: claim [src]`. The register = the source's
strength. The draft can claim nothing that isn't here.

**WRITE `output/outline.md`:**
- Beats in order: claim → reason → evidence → strongest objection → reply,
  matched to the format's parts and the D-list. Each beat lists its F-tags.
- A `## Imagery` block: ONE controlling metaphor from the audience's world,
  supporting analogies in the same domain, each mapped to a beat. Ask of each:
  *would this audience use this image among themselves?* If no, cut it.
- Per-beat visual mark only where a chart/diagram/table beats prose, and only if
  the dial allows.

**CHECK before STOP:**
- Every D-item → at least one beat. ✓
- Every fact has an S-register. ✓
- One metaphor, audience's world, passes the condescension test. ✓
- No image or fact that isn't from research / the audience's world. ✓

**Anchors (restate at the end of your work):** goal = `{from brief}`; D-list =
`{from brief}`; audience = `{from brief}`. The outline is the contract stage 03
obeys exactly.
