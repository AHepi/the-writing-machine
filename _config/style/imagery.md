# imagery.md — the repertoire rules for metaphor, analogy, and meme

*Read at stage 02 (planning the `## Imagery` block) and stage 03 (writing the
images the outline licensed). Imagery is a window's vivid scene, not decoration
bolted on — classic style uses it freely, but on rules. Nothing off the outline's
imagery map may appear in the draft.*

---

## 1. One controlling metaphor per piece

Choose a single governing image and draw it **from the audience's own world**.
The audience file §3 tells you that world.

- The **comic fan** gets continuity, retcons, canon, the multiverse.
- The **child** gets playground physics, sharing, fairness, games with rules.
- The **socialist** gets labor, solidarity, who-owns-what, the commons.
- The **conservative** gets stewardship, inheritance, institutions, the
  household.

The controlling metaphor is named in the outline's `## Imagery` block and mapped
to the beat it opens.

## 2. Supporting images must be complementary

Every supporting analogy lives in the **same source domain** as the controlling
metaphor, or in a domain explicitly declared compatible in the imagery block.
Complementarity is a rule, not a hope. A piece whose controlling metaphor is a
garden does not suddenly reach for a courtroom — unless the imagery block
declared garden+seasons+weather as the compatible set up front.

A metaphor from one domain and a supporting image from an unrelated one is a
**mixed metaphor** — a stage 04 defect.

## 3. Layer the metaphor for complex topics

For a hard subject the controlling metaphor is **layered**, not multiplied:

- Introduce it at its **simplest** early, where the argument is still simple.
- **Extend** it — never replace it — as the argument's complexity grows.
- It should **unfold with the text**, gaining structure exactly as the idea
  does, so the reader who holds the early image already holds the scaffold for
  the late one.

Replacing the metaphor midway is abandonment; extending it is craft.

## 4. The condescension test — gates everything

Before any image ships, ask: **would a member of this audience use this image
among themselves?**

- Yes → it respects them; license it.
- No, it's an outsider's idea of how they talk → cut it. It patronizes.

This is the imagery-side enforcement of the respect floor in `rules.md`. An
analogy that explains the audience to itself in a stranger's voice fails even if
it is technically apt.

## 5. Memes

- Licensed by **format and audience jointly** — Reddit yes, op-ed no; comic-fan
  yes, economist no. Both must permit it.
- **Evergreen over topical** — a meme that will read as dead in six months
  dates the piece. Prefer the durable.
- **Listed on the imagery map** like every other image. A meme that isn't in the
  outline's imagery block may not appear in the draft.

## 6. The map is the contract

The outline's `## Imagery` block lists the controlling metaphor, the compatible
domains, every supporting image mapped to its beat, and any licensed meme. Stage
03 renders **exactly that set and nothing else**. Stage 04 flags any image in
the draft that isn't on the map (orphaned) and any image on the map rendered in
the wrong domain (mixed). Generation is downstream of planning, always.
