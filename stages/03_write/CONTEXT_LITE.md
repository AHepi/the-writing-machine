# Stage 03 — Write (LITE)

*Pre-chewed contract for thrift-ON models. Same job as `CONTEXT.md`. Write the
draft the outline licensed — no new facts, no new images, no new visuals — and
cross the D-gate before you stop.*

**READ** (in order):
1. `../00_brief/output/brief.md` — goal, audience, format, length, D-list,
   provocation.
2. `../02_compile/output/factsheet.md` — the ONLY facts you may use (each has an
   S-register; don't claim above it).
3. `../02_compile/output/outline.md` — beats + `## Imagery` + visual marks. This
   is your whole scope.
4. `_config/style/pinker-core.md` — the prose floor.
5. `_config/style/imagery.md`, and `visuals.md` if there are visual marks.
6. The audience file and the format file named in the brief.

**WRITE `output/draft.md`:**
- One beat at a time, in order, using only that beat's F-tags. (Tier C / lazy
  models: finish and verify each beat before starting the next.)
- Controlling metaphor: simplest first, extended (never swapped) as you go. Only
  images on the map.
- Visual marks → typed specs (`CHART:`/`DIAGRAM:`/`TABLE:`) with real F-tag
  `data:` and a one-sentence `message=`, beside the claim. Never "insert graph
  here."

**FORBIDDEN phrases:** "in summary," "the remaining sections follow,"
"similarly," "and so on." Last third at full density. No compressing later
beats.

**CHECK — the completion ritual (reprint, don't assert):**
Reprint the full D-list. Beside each item, quote one line from your own draft
that satisfies it. No quote = not done.
- D1…D<n>: each quoted. ✓
- Length in band (state count). ✓
- Every claim → an F-tag, none above its S-register. ✓
- Every image on the map; one consistent metaphor. ✓
- No forbidden phrase. ✓

**Anchors (restate at top AND bottom of your work):** goal = `{from brief}`;
full D-list = `{from brief}`; audience register = `{from audience file §3}`.
Cross the finish line: a draft with an unquoted D-item is unfinished.
